# Документация по облигациям (Bonds)

[← Вернуться к README](../README.md)

Здесь собраны основные поля, которые возвращает MOEX ISS API для рынка облигаций, и их соответствие типам в `pymoex`. Посмотреть все поля можно тут: https://iss.moex.com/iss/engines/stock/markets/bonds/

## Поля используемые в Московской бирже
### Securities (Инструменты)
Статические параметры выпуска (меняются редко).

| Поле (API) | Тип в Python | Описание |
| ----------- | ----------- | ----------- |
| **Идентификация** |  |  |
| `SECID` | `str` | Идентификатор финансового инструмента |
| `BOARDID` | `str` | Идентификатор режима торгов |
| `ISIN` | `str` | ISIN |
| `SHORTNAME` | `str`  | Краткое наименование ценной бумаги |
| `SECNAME` | `str` | Наименование финансового инструмента |
| `REGNUMBER` | `str` | Регистрационный номер |
| `STATUS` | `str` | Статус |
| `LISTLEVEL` | `int` | Уровень листинга |
| `SECTYPE` | `str` | Тип ценной бумаги |
| `BONDTYPE` | `str` | Вид облигации |
| `BONDSUBTYPE` | `str` | Подвид облигации |
| **Параметры номинала** |  |  |
| `FACEVALUE` | `Decimal` | Непогашенный долг |
| `FACEUNIT` | `str` | Валюта номинала |
| `CURRENCYID` | `str` | Валюта, в которой проводятся расчеты по сделкам |
| `LOTSIZE` | `int` | Размер лота |
| `LOTVALUE` | `Decimal` | Номинальная стоимость лота, в валюте номинала |
| `MINSTEP` | `Decimal` | Мин. шаг цены |
| `ISSUESIZEPLACED` | `int` | Количество ценных бумаг в обращении |
| **Купоны и доходность** |  |  |
| `COUPONVALUE` | `Decimal` | Сумма купона, в валюте номинала |
| `COUPONPERCENT` | `Decimal` | Ставка купона, % |
| `ACCRUEDINT` | `Decimal` | НКД на дату расчетов, в валюте расчетов |
| `NEXTCOUPON` | `date` | Дата окончания купона |
| `COUPONPERIOD` | `int` | Длительность купона |
| **Календарь и цены** |  |  |
| `MATDATE` | `date` | Дата погашения |
| `BUYBACKDATE` | `date` | Дата, к кот.рассч.доходность |
| `BUYBACKPRICE` | `Decimal` | Цена оферты |
| `OFFERDATE` | `date` | Дата Оферты |
| `PREV` | `Decimal` | Цена последней сделки предыдущего торгового дня, % от номинала |
| `PREVPRICE` | `Decimal` | Цена последней сделки пред. дня, % к номиналу |
| `PREVWAPRICE` | `Decimal` | Средневзвешенная цена предыдущего дня, % к номиналу |
| `YIELDATPREVWAPRICE` | `Decimal` | Доходность по оценке пред. дня |

### Marketdata (Ход торгов)
Динамические данные, обновляемые в реальном времени в ходе сессии.

| Поле (API) | Тип в Python | Описание |
| ----------- | ----------- | ----------- |
| **Цены (% от номинала)** |  |  |
| `LAST` | `Decimal` | Цена последней сделки, % |
| `OPEN` | `Decimal` | Цена первой сделки, % к номиналу |
| `LOW` | `Decimal` | Минимальная цена сделки, % к номиналу |
| `HIGH` | `Decimal` | Максимальная цена сделки, % к номиналу |
| `WAPRICE` | `Decimal` | Средневзвешенная цена, % к номиналу |
| `LCLOSEPRICE` | `Decimal` | Цена закрытия |
| **Спрос и предложение** | | |
| `BID` | `Decimal` | Цена спроса (котировка на покупку) на момент окончания торговой сессии, % от номинала |
| `OFFER` | `Decimal` | Цена предложения (котировка на продажу) на момент окончания торговой сессии, % от номинала |
| `SPREAD` | `Decimal` | Разница между лучшей котировкой на продажу и покупку (спред), % к номиналу |
| `BIDDEPTHT` | `int` | Совокупный спрос |
| `OFFERDEPTHT` | `int` | Общий объем котировок на продажу, лотов |
| **Активность и объемы** | | |
| `NUMTRADES` | `int` | Количество заключенных сделок, штук |
| `VOLTODAY` | `int` | Объем заключенных сделок в единицах ценных бумаг, штук |
| `VALTODAY` | `Decimal` | Объем в валюте, в которой проводятся расчеты по сделкам |
| `QTY` | `int` | Объем последней сделки, лотов |
| `VALUE` | `Decimal` | Объем последней сделки, руб. |
| **Расчетные показатели** | | |
| `YIELD` | `Decimal` | Доходность по последней сделке |
| `EFFECTIVEYIELD` | `Decimal` | Эффективная доходность |
| `YIELDATWAPRICE`| `Decimal` | Доходность по средневзвешенной цене, % годовых |
| `DURATION` | `int` | Дюрация, дней |
| `LASTCHANGE` | `Decimal` | Изменение цены последней сделки к цене последней сделки предыдущего торгового дня |
| **Служебная информация** | | |
| `TRADINGSTATUS` | `str` | Состояние торговой сессии |
| `UPDATETIME` | `time` | Время последнего обновления |
| `SYSTIME` | `datetime` | Время загрузки данных системой |

## Пример использования в коде
Благодаря типизации, работа с полями выглядит так:

```python
# Получаем данные по облигации
bond = await client.bond("RU000A10DS74")

print(f"Название: {bond.short_name}")
print(f"Текущая цена: {bond.last_price}%")
print(f"Доходность: {bond.effective_yield}%")
print(f"Ближайший купон: {bond.next_coupon} ({bond.coupon_value} {bond.face_unit})")
```
